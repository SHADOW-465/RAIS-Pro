CORE

What MO!D is

MO!D turns the quality paperwork Disposafe already produces (rejection registers, inspection reports, messy Excel) into trustworthy, traceable, money-denominated operational intelligence — making their existing Lean practices automatic and continuous. Not a dashboard; an operational truth engine: it reconstructs what actually happened from imperfect records, proves every number, and asks a human when reality is ambiguous.

Compliance frame

turns features into audit assets
Medical device, not drugCONFIRMED
Disposafe = disposable medical devices → ISO 13485 QMS + Medical Device Rules 2017 (CDSCO; MD-5/MD-9 by class). NOT drug Schedule M. Confirm device class at the meeting.
ALCOA+ / Part 11BUILT-IN
Immutable ledger + who/what/when on every action + cell-level traceability = Attributable, Original, Accurate by design. Auditors open the audit trail first.
audit-readiness as a side effect of using the product.
CAPA-shaped outputV2
Recommendations emitted as CAPA records (problem → root cause → corrective + preventive → owner → due → effectiveness). CAPA is a mandatory ISO 13485 artifact — MO!D becomes compliance evidence, not just analytics.
Validation boundaryFLAG
As decision-support, light validation burden. If MO!D becomes system-of-record → CSV / GAMP-5 scope. Flag early, price it later.
