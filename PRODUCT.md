# Product

## Register

product

## Users

Plant operators, quality engineers, QA managers, and GMs at a regulated medical-device manufacturer (Disposafe FBC line). They work under shift pressure, anomaly chase, audit risk, and monthly stewardship — not leisure exploration. Primary jobs: capture daily quality facts honestly; investigate rejection spikes to gate/size/defect/batch/source; own CAPA; prove numbers to auditors (ISO 13485 / CDSCO).

## Product Purpose

MO!D / RAIS-Pro is an **operational evidence system**: capture (entry-first + Excel bulk) → verify/adjudicate → append-only ledger → deterministic analytics → decision support → ALCOA+ audit proof. AI classifies and narrates only; the model never invents KPI maths. Success is trustworthy plant numbers that are investigable, money-denominated, and exportable — not chart density.

## Brand Personality

Operational · trustworthy · plain-spoken. Editorial manufacturing intelligence (Linear / Stripe Dashboard / Vercel register density), not consumer delight or BI theater. Hierarchy from size/weight/spacing; burnt-orange accent for status only.

## Anti-references

- Glassmorphism / stock-market dashboard clichés  
- Feature-inventory sidebars as product architecture  
- Silent “smart” data fixes in regulated contexts  
- Fake demo KPIs / invented AI numbers  
- Chart.js / lucide / framer-motion aesthetic (explicitly removed)

## Design Principles

1. Trust is the interface — every number is traceable; success means ledger events  
2. One operational question per surface, in plant language  
3. Progressive disclosure along status → locus → cause → integrity → cost → action → proof  
4. Role-shaped surfaces over one ledger; capture UX is first-class  
5. Growth as depth inside existing questions, not new top-level destinations by default

## Accessibility & Inclusion

WCAG AA target for interactive app UI. Tabular numerals for data. Legible body ≥14px (product convention). Reduced-motion respect for pure CSS animations. Dual-audience Explain layer for GM vs QE vocabulary on the same facts. Hindi labels for operator entry where product supports them.
